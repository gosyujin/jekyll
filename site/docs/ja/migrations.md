もし、他のブログから Jekyll に移行する場合、 Jekyll のインポーターは
あなたの移行の手助けをしてくれます。
あなたのサイトを Jekyll にインポートすることについては、
[`jekyll-import` docs site](http://import.jekyllrb.com) で詳しく知ることができます。
